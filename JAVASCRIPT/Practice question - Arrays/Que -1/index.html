Que1. For a given array with marks of students ->[ 85, 97, 44, 37, 76, 60 ], find the avarage marks of the entire class.


Ans :
let marks = [ 85, 97, 44, 37, 76, 60 ];
let sum = 0;
for (let val of marks){
console.log(val);
sum += val;
}
let avg = sum /marks.length;
console.log(`avg marks of the class = ${avg}`);
